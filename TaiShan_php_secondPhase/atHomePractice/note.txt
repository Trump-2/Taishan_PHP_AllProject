網頁檔案之間在 php 裡無法完全都使用 post，第一個頁面傳送時用 post，那第二個頁面傳回來時，第一個頁面就要用 get 接


get / post 僅限於兩個網頁檔案之間資料的傳遞，用於多個網頁時就會顯得麻煩

css property [ justify-content]：
    1.space-around => 容器裡有兩個物件時，有 4 個空白；三個物件時，有 6 個空白，依此類推；空白數量的公式: 2n
    2.space-between => 容器裡有兩個物件時，有 1 個空白；三個物件時，有 2 個空白，依此類推；空白數量的公式: n - 1
    3.space-evenly  => 容器裡有兩個物件時，有 3 個空白；空白數量的公式: n + 1

檔案之間的傳值，除了可透過 html 的 form 表單外，也可透過 html 的 < a >

要在 html 中使用 php 裡宣告的變數，要使用【 <?= 變數名稱 ?> 】，而 <?= 相當於 【<?php echo】
